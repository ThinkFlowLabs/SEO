# Medical Travel Colombia, SEO Audit and Growth Strategy

Prepared by **ThinkFlowLabs**, June 2026. Confidential, prepared for medicaltravelcolombia.com.

## The one-line story
The site ranks top-three for "medical tourism colombia" (a low-volume term) while sitting on ~47,000 yearly impressions for "veneers in Colombia" and its variants that it converts almost none of, because ten pages cannibalize each other, the flagship page carries live placeholder text, and the domain has almost no authority. Consolidate the dental cluster, fix the cleanup items, and build cost/safety content to capture demand that already exists.

## Top findings
1. Ten dental/veneer pages cannibalize each other; none reaches page one (~98k combined impressions, ~168 clicks).
2. Live lorem-ipsum testimonials on the 43,839-impression flagship `/landing-colombian-veneers/`.
3. Authority Score 7; backlinks dominated by a Moldova spam cluster; one legitimate medical link.
4. No medical/FAQ/Review schema; H1 chaos; English content tagged `lang="es"`.
5. `/contact-us/` 404s while the nav links to it; impression base eroding.

## Method
GSC 16-month export + Playwright crawl (15 templates, full-page) + DataForSEO + Semrush + Perplexity/Brave + a 5-prompt x 4-model flagship LLM citation test (Claude Sonnet 4.6, Gemini 3.1 Pro, GPT-5.5, Sonar-Reasoning-Pro). MTC cited 4 of 20, all on the Cartagena-veneers wedge and the head term.
